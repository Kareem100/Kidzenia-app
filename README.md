# Kidzenia-app

### App Description 

An Easily Animated App for Kids. </br> 
For learning (English Alphabet, Numbers, Arabic Alphabet and Numbers). </br>
In Addition to Some Entertainment Educational Videos. </br> 

***

##### Tools Used

- Adobe After Effects
- GIFs
- Arabic Locale Translation
